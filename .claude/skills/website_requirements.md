## Адаптивность (Mobile First)
- Сайт должен корректно отображаться на всех устройствах: от 320px (iPhone SE) до 1920px (4K).
- Использовать медиа-запросы или классы Tailwind (sm:, md:, lg:, xl:).
- На мобильных:
  - меню становится гамбургером (если есть навигация)
  - отступы не более 16px по бокам
  - кнопки имеют минимальную высоту 44px (для тапа)
  - шрифты не меньше 14px для основного текста
- Проверять адаптивность через эмуляцию устройств в Chrome DevTools (Ctrl+Shift+M) на всех этапах разработки.
- Lighthouse mobile должен быть ≥90 по Performance и Accessibility.
