# Визуальная система для сайта «ПошивНН»

## Цвета
- Основной: тёмно-серый/чёрный (премиум, индустриальный стиль) — `#1E1E1E` для фона шапки/подвала.
- Акцент: белый (`#FFFFFF`) или светлый серый (`#F5F5F5`) для контраста.
- Элементы действия: глубокий тёмно-синий (`#0A2647`) или бордовый (`#8B0000`) для кнопок — по желанию.
- Для текста: на светлом фоне — `#111`, на тёмном — `#DDD`.

## Шрифты
- Основной: Inter или Roboto (Google Fonts).
- Акциденции: Montserrat (заголовки H1, H2).
- Кегли: заголовки – 2rem–2.5rem на десктопе, адаптивно.

## Компоненты
- Кнопки: скругление 4–6px, минимальная высота 44px (для тапа).
- Карточки портфолио: тень, переход при наведении, соотношение сторон 1:1.
- Плавающая кнопка: круг 56px, градиент, иконка мессенджера (лучше две иконки).

## Адаптивность
- На мобильных: колонка 1fr, отступы 16px.
- На планшетах: колонка 2fr.
- На десктопе: максимальная ширина контента 1200px, margin: auto.

## Анимация
- Плавное появление секций при скролле (опционально, но не обязательно).
- Hover на кнопках: смена цвета/тени.
